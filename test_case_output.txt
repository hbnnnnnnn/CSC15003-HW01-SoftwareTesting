| TC ID | Objective                                | Input                           | Steps                                                                    | Expected                                                                | Actual | Verdict |
| ----- | ---------------------------------------- | ------------------------------- | ------------------------------------------------------------------------ | ----------------------------------------------------------------------- | ------ | ------- |
| TC01  | Verify fan power-on functionality        | Fan connected to power          | 1. Plug in fan.<br>2. Press Power button.                                | Fan turns on and blades rotate.                                         |        |         |
| TC02  | Verify fan power-off functionality       | Fan running                     | 1. Turn fan on.<br>2. Press Power button again.                          | Fan stops rotating completely.                                          |        |         |
| TC03  | Verify Speed Level 1                     | Speed setting = 1               | 1. Turn on fan.<br>2. Select Speed 1.                                    | Fan operates at lowest speed.                                           |        |         |
| TC04  | Verify Speed Level 2                     | Speed setting = 2               | 1. Turn on fan.<br>2. Select Speed 2.                                    | Airflow increases compared to Speed 1.                                  |        |         |
| TC05  | Verify Speed Level 3                     | Speed setting = 3               | 1. Turn on fan.<br>2. Select Speed 3.                                    | Fan operates at maximum speed.                                          |        |         |
| TC06  | Verify speed switching                   | Speed levels 1, 2, 3            | 1. Turn on fan.<br>2. Change speeds sequentially.                        | Fan changes speed correctly without interruption.                       |        |         |
| TC07  | Verify oscillation ON                    | Oscillation enabled             | 1. Turn on fan.<br>2. Push oscillation knob/button.                      | Fan head rotates left and right continuously.                           |        |         |
| TC08  | Verify oscillation OFF                   | Oscillation active              | 1. Enable oscillation.<br>2. Disable oscillation.                        | Fan head remains fixed in one direction.                                |        |         |
| TC09  | Verify vertical tilt adjustment          | Different tilt angles           | 1. Loosen tilt lock.<br>2. Adjust fan head up/down.<br>3. Tighten lock.  | Fan head stays at selected angle.                                       |        |         |

| TC12  | Verify noise level acceptability         | Maximum speed                   | 1. Run fan at highest speed.<br>2. Listen for abnormal sounds.           | No unusual rattling, grinding, or squeaking sounds.                     |        |         |

**Verdict values** are typically:

* **Pass** – Expected result matches Actual result.
* **Fail** – Expected result does not match Actual result.
* **Blocked** – Test could not be executed due to an issue.
* **N/A** – Not applicable.
